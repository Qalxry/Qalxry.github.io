图神经网络（Graph Neural Networks）

GCN（Graph Convolutional Network）

GCN是一种基于图的神经网络，它通过在图的数据上进行卷积操作来学习节点之间的关系和特征。

其主要特点包括:

  1. 利用图结构进行卷积运算。GCN可以在图结构上直接进行卷积操作，充分利用节点之间的连接关系。
  2. 学习节点的表示。GCN可以学习每个节点的特征表示，通过聚合邻居节点的信息来更新每个节点的表示。
  3. 层次聚合节点信息。GCN通过多层图卷积网络层次聚合节点邻居的信息，学习节点的高阶连接信息。
  4. 可以处理不同规模和类型的图。GCN建模方式可以自然地扩展到不同规模和类型的图上。
  5. 应用广泛。GCN已在许多领域获得应用，如社交网络分析、知识图谱 completion、分子指纹学习等。

总的来说，GCN通过图卷积运算高效聚合节点邻居信息，可以有效学习图上的节点表示和结构知识，是图神经网络中重要的一类模型。

TGC（Temporal Graph Convolutional Network）

TGC 是 GCN 的一个时间图扩展版本。

TGC 的主要特点是:

  1. 可以处理动态时间图数据。TGC可以处理图结构随时间变化的动态图数据。
  2. 利用时间卷积网络。TGC利用Temporal Convolutional Network来捕获时间维度上的图模式。
  3. 学习时间依赖的节点表示。TGC可以学习每个时间步的节点表示，并建模节点表示在时间上的依赖关系。
  4. 添加时间注意力机制。TGC中可以加入时间注意力机制，让模型自动关注不同时间步的重要性。
  5. 应用包括预测链接形成、预测节点属性等。TGC可用于 Time-varying graph上的预测任务。
  6. 与GCN相比，TGC通过建模时间信息，可以更好地处理动态变化的图数据。

综上，TGC是GCN的重要扩展，可以处理时间变化的图，学习时间依赖的节点表示，在许多时间相关的图学习任务中展示了很好的效果。

TGC的提出和发展:

  1. 最初的TGC模型是在2017年由Pham等人在论文《Convolutional Neural Networks over Tree Structures for Programming Language Processing》中提出。
  2. 然后在2018年，Zhang等人在论文《GaAN: Gated Attention Networks for Learning on Large and Spatiotemporal Graphs》中提出了时空图注意力网络GaAN，这也是TGC模型的早期代表。
  3. 2019年，论文《Temporal Graph Convolutional Networks for Predicting Traffic Speed and Density》将TGC用于交通速度和密度预测。
  4. 2020年，Lei等人在论文《TGAT: Temporal Graph Attention Network for Traffic Forecasting》中提出了TGAT模型。
  5. 2021年，论文《Temporal Graph Networks for Deep Learning on Dynamic Graphs》对TGC模型做了系统总结。

综上，TGC模型最早于2017年被提出，经过几年的发展，已成为动态图学习的重要模型。主要的论文发表在ACL、IJCAI、AAAI等顶会上，是图神经网络和时序预测的重要研究方向。

夏普比率（Sharpe Ratio）

定义：

夏普比率（Sharpe Ratio）是一种金融指标，用于衡量投资组合或资产的风险调整收益率。

用于度量每承担一单位的风险所获得的超额报酬（相对无风险资产)。

其中：

  • 是投资组合或资产的预期收益率。
  • 是无风险利率，通常使用短期国债收益率来代表，因为它被认为是无风险投资的收益率。
  • 是投资组合或资产的标准差（收益波动率），代表其波动性（风险）。
举例：

假如国债的回报是4%，而您的投资组合预期回报是16%，您的投资组合的标准差（收益波动率）是6%。 那么用16%-4%可以得出12%（代表您超出无风险投资的回报)，再用12%÷6%=2，代表投资者风险每增长1%，换来的是2%的多余收益。

说明：
  • 该比率是由美国经济学家威廉·夏普（William F. Sharpe）于1966年提出的，因而得名。
  • 夏普比率主要用于评估投资组合或资产的表现，考虑了投资的风险水平。它衡量的不仅仅是收益率的高低，还考虑了投资的波动性（风险）。夏普比率越高，表示每承担一单位的风险，可以获得更高的超额收益。
  • 夏普比率的数值越高，意味着每单位风险获得的超额收益越高，这被认为是更好的表现。因此，投资者可以利用夏普比率来比较不同投资组合或资产的表现，并选择那些风险调整收益率较高的投资。
  • 需要注意的是，夏普比率有其局限性，它假设收益率服从正态分布，而市场中的实际收益率可能不服从这种分布。此外，该指标只考虑了投资的波动性风险，而忽略了其他可能的风险因素。因此，在使用夏普比率时，投资者应该结合其他指标和风险管理方法来做出更全面的投资决策。

基于LSTM模型的智能选股策略

如图所示，LSTM选股策略构建包含下列步骤：
  • 获取数据 ：A股所有股票。
  • 特征和标签提取 ：计算7个因子作为样本特征；计算5日个股收益率，极值处理。
  • 特征预处理 ：进行缺失值处理；去掉特征异常的股票，比如某个特征值高于99.5%或低于0.5%的；标准化处理，去除特征量纲/数量级差异的影响。
  • 序列窗口滚动 ：窗口大小设置为5，滚动切割。这里的意思是使用过去5天的因子数据作为输入。窗口大小可调整，在“序列窗口滚动”模块中进行。
  • 搭建LSTM模型 ：构建两个隐含层的LSTM长短期记忆神经网络预测股票收益率（回归模型）。在可视化策略中表现为1个输入层；一个LSTM和一个全连接层作为隐藏层，每构建一层进行dropout断开一些神经元防止过拟合；最后一个全连接层作为输出层（输出维度调整为1）。
  • 模型训练与预测 ：使用LSTM模型进行训练和预测；可以尝试多种激活函数，策略默认为tanh。
  • 策略回测 ：利用2010到2016年数据进行训练，预测2016到2019年的股票表现。每日买入预测排名最靠前的30只股票，至少持有5日，同时淘汰排名靠后的股票。具体而言，预测排名越靠前，分配到的资金越多且最大资金占用比例不超过20%；初始5日平均分配资金，之后，尽量使用剩余资金（这里设置最多用等量的1.5倍）。
  • 模型评价 ：查看模型回测结果。
LSTM模型的参数如下：

  • 输入数据：7个因子，使用了过去5天的因子数据，因此输入7*5的一个矩阵。

  • LSTM层：激活函数采用tanh，recurrent激活函数采用hard_sigmoid。循环核初始化方法Orthogonal，权值使用glorot_uniform初始化方法，偏置向量使用Zeros初始化方法。

  • 全连接层：激活函数tanh。权重使用glorot_uniform初始化方法，偏置向量使用Zeros初始化方法。

  • 输出层：最后一个全连接层。需要选择“输出空间维度”为1，因为要得到个股的收益率预测结果，这是一个值。

  • dropout：0.2。在不同隐藏层之间使用dropout可以让网络更耐用并且避免过拟合。

  • 训练次数率：epochs值为5，共训练5轮，以mae作为评估指标。

  • 优化器：RMSProp。

  • 损失函数：均方误差MSE。

Winsor化方法

Winsor化是一种常见的处理极值的方法，主要步骤如下:

  1. 对样本数据进行排序

  2. 设定上下百分位截断点，例如上5%和下5%

  3. 将上5%的数据替换为95%分位数的值

  4. 将下5%的数据替换为5%分位数的值

  5. 中间90%的数据保持不变

这样通过截断极端值来减少异常数据的影响，使得特征分布更加稳定。

举个例子，对以下收益率数据进行Winsor化处理:

原数据：[-5%， 1%， 6%， -2%， 4%， 15%， 3%， 10%， -4%， 2%]

排序后：[15%， 10%， 6%， 4%， 3%， 2%， 2%， 1%， -2%， -4%， -5%]

设定上下5%分位数截断，则:

Winsor化后：[6%， 6%， 6%， 4%， 3%， 2%， 2%， 1%， -2%， -2%， -2%]

Winsor化的参数也可以根据实际情况进行调整，如10%或20%分位数截断等。这个方法可以有效控制异常值对模型的影响。

通过所有股票的未来可能的盈利率进行排序来选择购入股票，是一种不错的idea，可以进行多品种交易。

某种想法：

  • 对所有股票的5、10、15、30、60天盈利率（或者更加细粒度）进行预测，然后根据日均盈利率（或者其他更好描述每日涨幅的指标）的排名来进行选股，这样达到了一定的变长持有的效果。这样做可能需要训练多个模型，并行推理。

股票预测的时间关系排名

引言

大多数现有的深度学习解决方案并没有针对投资目标进行优化，即选择预期收益最高的最佳股票。具体来说，他们通常将股票预测表述为分类（预测股票趋势）或回归问题（预测股票价格）。更重要的是，他们将股票视为彼此独立的。实际上股票（或公司）之间的丰富关系中的有价值的信号，例如两只股票属于同一板块以及两家公司具有供应商-客户关系，没有被考虑。

方法

  • 通过LSTM获取股票t日的顺序嵌入。
  • 对股票关系进行建模（同行业、同部门、同板块、上下游、供应链），并通过多热向量的时间感知关系强度函数进行图权重调整。调整后的多热向量通过时序图卷积获得关系嵌入。
  • 对顺序嵌入和关系嵌入进行concat，传入全连接层得到未来1日盈利率。
  • 通过预测的未来1日盈利率对股票进行排序，选取排名Top1、5、10的股票（三种策略）进行买入，第二日卖出。
架构

关键点

  • 使用了股票之间的关系Embedding对时序信息进行增强或修正，捕捉相似股票之间的变化相同性，使得预测结果更加准确。
  • 创新性地使用TGC（Temporal Graph Convolution）（对比GCN）

结果

性能优于最先进的股票预测解决方案，在纽约证券交易所和纳斯达克分别实现了 98% 和 71% 的平均回报率。

Claude：深度学习金融SOTA模型

Qalxry
请你提出有关金融市场中深度学习目前最为先进的模型

Claude：

对于金融市场，深度学习当前最先进的模型主要有以下几类:

  1. 基于卷积神经网络(CNN)的模型:利用CNN提取金融时间序列中的局部特征和全局特征，并用于预测股票价格、市场趋势等。例如Temporal Convolutional Network (TCN)。

  2. 基于循环神经网络(RNN)的模型:利用RNN模型历史序列信息，处理金融时间序列问题。例如长短期记忆网络(LSTM)、门控循环单元(GRU) commonly used.

  3. 注意力机制模型:加入了注意力机制的RNN模型，可以关注金融时间序列的关键部分。例如带Attention的LSTM/GRU模型。

  4. 图神经网络(GNN):通过建模金融市场参与者之间的关系，进行股票/市场预测。例如GCN，GAT等。

  5. 强化学习模型:将金融交易建模为标杆环境，以实现最优投资组合等。例如A2C、DDPG等算法。

  6. Transformer类模型:利用自注意力机制有效模型长期依赖。例如时间嵌入的Transformer，可以处理金融序列数据。

  7. 多模态模型:融合文本、图结构、时间序列等不同模式数据，提高了金融预测的效果。

  8. 生成对抗网络(GAN):利用GAN的生成过程进行金融序列的模拟，辅助预测。

随着深度学习在金融界的广泛应用，相信会有更多创新型模型应运而生。但 Transformer类模型目前看来是较为前沿和普适的选择。

Qalxry
你知道在这个领域上（金融市场中深度学习）的最有名的模型吗（最好是最近5年内的）

Claude：

是的，最近5年来，在金融市场深度学习领域比较有名和前沿的模型有:

  1. Transformer - 最近几年非常热门的自注意力机制模型，被广泛应用到时序数据预测，如股票价格预测。代表性工作如Similarity Embedded Temporal Transformer (SETT)。
  2. BERT - Google推出的语言模型BERT也被应用到金融文本情感分析等任务上，表现不错。例如FinBERT就是针对金融领域做了预训练的BERT模型。
  3. Graph Neural Networks (GNN) - 利用图网络建模金融市场，进行股价预测。典型的工作如Graph Attention Networks for Stock Price Prediction等。
  4. Deep Reinforcement Learning (DRL) - DRL让机器可以像人类交易员那样通过试错进行自动化交易。例如基于A2C、PPO的算法。
  5. TCN - 利用一维卷积网络提取金融序列的临近关联特征。例如在高频交易算法中广泛使用。
  6. LSTM/GRU - 传统RNN模型，但由于能够捕捉长时依赖性，在金融预测中仍大量被使用。
  7. Wavenet - Google提出的生成模型，使用于模拟连续金融时间序列。
  8. GAN - 对抗生成网络，能够对数据分布进行建模，然后生成更真实的样本。用于金融数据的模拟。

上述模型都取得了不错的实际效果，使得深度学习在金融预测和投资决策中越来越重要。

Qalxry
继续，还有更加sota的模型吗

Claude：

好的，除了之前提到的模型，近年来金融领域深度学习中也出现了一些更前沿的模型和方向:

  1. Temporal Fusion Transformer (TFT) - 一种集成注意力机制与卷积网络的Transformer模型，专门用于处理时间序列，在金融预测任务中表现突出。
  2. Informer - 全称“Inference-Dimension Reduction-Information Accumulator”，使用注意力机制的Transformer模型，在时间序列预测效果优于传统RNN。
  3. Flow Models - 利用正常分布的流模型(Normalizing Flows)对金融时间序列建模，可以更好地刻画其分布，广泛用于量化交易。
  4. Deep Calendar Networks - 将日历信息融入深度学习模型中，这类模型在进行金融时间序列建模时，考虑了日期的辅助信息。
  5. Capsule Networks - 利用胶囊网络学习金融数据的层次关系，进行股票或加密货币价格预测。
  6. Federated Learning - 在保护用户隐私和数据权限的前提下，进行分布式协作式的金融模型训练。

  • 直接取斐波那契数列的某一段作为位置索引的嵌入向量。

具体来说,可以考虑以下实现方式:

  1. 设置LSTM的层数,例如总共3层。
  2. 每层LSTM的循环周期长度分别为:1, 1, 2, 3, 5, 8,...依次取斐波那契数列。
  3. 对于我们的3层LSTM,周期长度设置为:
  • 第1层:周期长度为1
  • 第2层:周期长度为1
  • 第3层:周期长度为2
  1. 在训练时,每层LSTM按照自己的周期长度进行循环计算。
  2. 循环计算得到的隐状态进行堆叠或者融合,作为该时间步的输出状态。

这种设计的理论依据是,不同的斐波那契周期长度可以捕捉文本中不同粒度的循环模式。高层可以模拟更长周期的循环规律。

一种多模型结合的编码器

对于时序信息，通过3种并行的方式来捕捉特征：CNN、RNN、Attention。

CNN可处理局部特征，RNN可处理连续性的长特征或周期特征，Attention可通过注意力来捕捉全局特征。

  • CNN可以用多种卷积核以及多种不同的改进性卷积方式（具有多跨度的卷积等）
  • RNN可以根据斐波那契数列对1、2、3、5、8......跨度进行不同的循环。
  • Attention可以应用多头、改进的注意力方式等各种方法。

这三种方式都要有一定的深度，以保证可以拟合复杂的特征函数。

对于3种方式得到的Embedding，可以进行Concatenate，这样就包含了各种特征的信息。

然后进行压缩，使特征更精简。
